# xframes-crystal

## Instructions

### Install Crystal

#### Windows

`winget install --id=CrystalLang.Crystal  -e`

### Running

#### Windows

`crystal run src/main.cr`

##### Notes

Copy the xframesshared files into the path returned by running this command

`crystal env CRYSTAL_LIBRARY_PATH`
